Shift Scope Analyser

generates an ECMAScript program's scope tree from a Shift AST

Demo

The program on the left is rendered on the right. Mousing over any identifier will highlight it all other identifiers which refer to the same variable, color-coded by type of access (declaration, read, or write).

let f = x(1); f(2); function x(x, y = () => x){ let {z} = {z:3}; return x => x + y() - z++; }

Installation

npm install shift-scope

Usage

import analyze from "shift-scope";
let globalScope = analyze(/* Shift AST */);
